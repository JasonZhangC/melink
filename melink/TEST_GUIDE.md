```bash
curl -X POST http://localhost:3000/api/share \
  -H "Content-Type: application/json" \
  -d '{"title":"黑客松交流","url":"http://localhost:3000/hackathon"}'
```


## 🐛 常见问题排查

### 1. 页面无法访问
- 检查开发服务器是否启动
- 检查端口3000是否被占用
- 检查控制台是否有错误信息

### 2. 图片无法显示
- 检查图片服务器是否运行（localhost:3845）
- 检查Next.js配置中的图片域名设置
- 检查网络连接

### 3. API调用失败
- 检查API路由文件是否正确创建
- 检查控制台是否有错误信息
- 检查网络请求是否正常

### 4. 样式显示异常
- 检查CSS文件是否正确导入
- 检查浏览器兼容性
- 检查CSS类名是否正确
